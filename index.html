actionHero is a minimalist transactional API framework for sockets and http clients

The actionHero API aims to simplify and abstract may of the common tasks that these types of APIs require. actionHero does the work for you, and he's not CRUD, and he's never taking a REST.

  • - actionHero is for the node.js server. Learn more here http://nodejs.org/
  • - actionHero is available in the NPM (node package manager). Learn more here
